# Design Description

## Concept

The flag must reflect light from all angles and be visible to both pedestrians and drivers. A cylindrical retroreflective flag was designed. The modular design allows easy transport and integration of other solutions.

## Components

### Bushing
- Cylindrical, 3 blocks
- Block 1: 3×M6 mounting holes + M20×2.5 thread
- Block 2: Access for nuts inside rover
- Block 3: 90 mm depth for internal fixation

### Adaptor 1
- Screws into Bushing (M20×2.5 thread)
- Connects to Adaptor 2 with 2×M3 screws
- Rounded section allows folding

### Adaptor 2
- Holds the circular reflective panel
- Connects to Adaptor 1 via two rectangular interfaces

### Retaining Ring
- Prevents self-unscrewing
- 2×M3 screws into Bushing + 1×M3 screw into Adaptor 1

### Compensator
- Vibration damping element
- Located between Adaptor 1 and Adaptor 2
- Material: TPU (recommended) or PETG

## Flag Dimensions

Based on human height (≈175 cm) and seated driver eye level (≈110 cm). Optimal viewing angle: 15–30° downward. Final height: **770 mm**.
