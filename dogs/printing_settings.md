# FDM Printing Settings (PETG)

## Recommended Settings

| Parameter | Value |
|-----------|-------|
| Nozzle temperature | 230–245°C |
| Bed temperature | 70–80°C |
| Layer height | 0.16–0.20 mm |
| Infill | **100%** |
| Wall lines | 4+ |
| Print speed | 40–50 mm/s |

## Print Orientation

| Part | Orientation |
|------|-------------|
| Bushing | Vertical (threads up) |
| Adaptor 1 | Vertical |
| Adaptor 2 | Horizontal |
| Retaining Ring | Flat on bed |
| Compensator | Flat on bed |

## Compensator (TPU)

| Parameter | Value |
|-----------|-------|
| Nozzle temp | 220–235°C |
| Bed temp | 50–60°C |
| Speed | 20–30 mm/s |

## Post-Processing

1. Remove supports
2. Tap M3 threads
3. Chase M20×2.5 thread with a tap
